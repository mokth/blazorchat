# Blazor Chat Application

A real-time chat application built with Blazor .NET 8 and SignalR featuring WhatsApp-style UI with support for text messages, voice messages, images, and file attachments.

## Features
- Real-time messaging with SignalR
- Modern WhatsApp-style UI
- Voice message support
- Image and file attachments
- Integration with ASP.NET WebForm 4.8
